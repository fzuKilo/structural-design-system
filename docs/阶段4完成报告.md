## 架构设计

### 类层次结构

```
StructureDrawer (抽象基类)
    ├── _get_structure_type() - 抽象方法
    ├── draw_plan() - 抽象方法
    ├── draw_elevation() - 抽象方法
    ├── draw_details() - 抽象方法
    └── generate_drawings() - 通用方法
        └── DrawingResults (数据类)

BeamDrawer (具体实现)
    └── 继承并实现StructureDrawer所有抽象方法

DrawerFactory (工厂类)
    ├── register() - 注册机制
    ├── create() - 创建方法
    ├── get_available_types() - 获取可用类型
    └── is_registered() - 检查注册状态

CADDrawingTool (Tool封装)
    └── 继承BaseTool，通过DrawerFactory路由
```

### 关键设计原则

1. **策略模式**: 不同结构类型的绘图逻辑通过策略模式分离
2. **工厂模式**: 使用DrawerFactory动态创建绘图器
3. **抽象优先**: 先定义抽象基类，再实现具体类
4. **无硬编码**: Tool代码中不硬编码具体结构类型

## 架构扩展性

### 添加新结构类型（如Frame）

只需以下步骤：

1. **创建FrameDrawer** (继承StructureDrawer)
```python
class FrameDrawer(StructureDrawer):
    def _get_structure_type(self) -> str:
        return "frame"

    def draw_elevation(self, design):
        # 实现框架立面图绘制
        pass
    # ... 其他方法
```

2. **在DrawerFactory中注册**
```python
DrawerFactory.register("frame", FrameDrawer)
```

3. **无需修改任何Agent代码** - 工厂模式自动路由
