## 实现组件（每个类型）

每个结构类型需要实现：

```
app/tool/
├── analyzers/
│   ├── {type}_analyzer.py        # 有限元分析
│   └── analyzer_factory.py       # 注册
├── drawers/
│   ├── {type}_drawer.py          # CAD绘图
│   └── drawer_factory.py         # 注册
└── evaluators/
    ├── {type}_evaluator.py       # 设计评估
    └── evaluator_factory.py      # 注册
```
