# 前端开发环境配置

## 1. 必需软件

### Node.js
- 版本：18.x 或更高
- 下载：https://nodejs.org/
- 验证安装：`node -v` 和 `npm -v`

### Git
- 用于代码同步
- 下载：https://git-scm.com/

## 2. 克隆项目

```bash
git clone https://github.com/Lin-0408-Yiran/structural-design-system.git
cd structural-design-system
git checkout dev
```

## 3. 前端依赖安装

```bash
cd frontend
npm install
```

### 主要依赖包（已在 package.json 中）
- Vue 3
- TypeScript
- Ant Design Vue
- Vue Router
- Pinia（状态管理）
- Axios
- Marked（Markdown 渲染）
- ECharts（图表）

## 4. 前端配置文件

### vite.config.ts
已配置代理，无需修改：
- API 代理：`/api` -> `http://localhost:8000`
- WebSocket 代理：`/ws` -> `ws://localhost:8000`

### 环境变量（可选）
创建 `.env.local`（如需自定义）：
```
VITE_API_BASE_URL=/api
```

## 5. 启动前端开发服务器

```bash
cd frontend
npm run dev
```

访问：http://localhost:5173
