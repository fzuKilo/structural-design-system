# GB 50017-2017 钢结构设计标准 (摘要)

## 第3章 基本设计规定

### 3.1 一般规定

3.1.1 钢结构应按承载能力极限状态和正常使用极限状态进行设计。

3.1.2 钢结构构件应进行下列承载能力极限状态计算:
- 强度计算
- 稳定性计算
- 疲劳计算
- 连接计算

### 3.2 挠度限值

3.2.1 受弯构件的挠度限值应符合下列规定:

| 构件类型 | 挠度限值 |
|---------|---------|
| 简支梁 | L/250 |
| 悬臂梁 | L/200 |
| 连续梁 | L/300 |
| 桁架 | L/250 |

其中 L 为构件的计算跨度。

## 第5章 受弯构件

### 5.1 一般规定

5.1.1 受弯构件应进行强度、整体稳定性和局部稳定性计算。

5.1.2 受弯构件的截面尺寸应满足下列要求:

**梁的最小截面尺寸:**
- 最小腹板厚度: 6mm
- 最小翼缘厚度: 8mm

**梁的高跨比:**
- 简支梁: 1/20 ~ 1/12
- 连续梁: 1/25 ~ 1/15
- 悬臂梁: 1/10 ~ 1/8

**梁的宽高比:**
- 工字形截面: 翼缘宽度/腹板高度 = 0.3 ~ 0.5
- 箱形截面: 宽度/高度 = 0.5 ~ 1.0

### 5.2 强度计算

5.2.1 受弯构件的抗弯强度应符合下列规定:

σ = M / Wn ≤ f

其中:
- σ: 弯曲正应力
- M: 弯矩设计值
- Wn: 净截面模量
- f: 钢材抗拉、抗压强度设计值

### 5.3 整体稳定性

5.3.1 受弯构件的整体稳定性应符合下列规定:

σ = βbM / Wx ≤ f

其中:
- βb: 等效弯矩系数
- Wx: 截面模量

### 5.4 局部稳定性

5.4.1 受弯构件的腹板和翼缘应满足局部稳定性要求:

**腹板高厚比限值:**
- 无加劲肋: h₀/tw ≤ 80√(235/fy)
- 有加劲肋: h₀/tw ≤ 150√(235/fy)

**翼缘宽厚比限值:**
- 外伸翼缘: b/t ≤ 15√(235/fy)

## 第7章 轴心受压构件

### 7.1 一般规定

7.1.1 轴心受压构件应进行强度和稳定性计算。

7.1.2 轴心受压构件的长细比应符合下列规定:

**长细比限值:**
- 主要受压构件: λ ≤ 150
- 次要受压构件: λ ≤ 200
- 柱间支撑: λ ≤ 200
- 屋盖支撑: λ ≤ 250

其中 λ = l₀/i (l₀为计算长度, i为截面回转半径)

### 7.2 强度计算

7.2.1 轴心受压构件的强度应符合下列规定:

σ = N / An ≤ f

其中:
- N: 轴向压力设计值
- An: 净截面面积
- f: 钢材抗压强度设计值

### 7.3 稳定性计算

7.3.1 轴心受压构件的稳定性应符合下列规定:

N / (φA) ≤ f

其中:
- φ: 轴心受压构件的稳定系数
- A: 截面面积

## 第8章 轴心受拉构件

### 8.1 一般规定

8.1.1 轴心受拉构件应进行强度计算。

8.1.2 轴心受拉构件的长细比不宜超过下列限值:
- 主要受拉构件: λ ≤ 350
- 次要受拉构件: λ ≤ 400

### 8.2 强度计算

8.2.1 轴心受拉构件的强度应符合下列规定:

σ = N / An ≤ f

其中:
- N: 轴向拉力设计值
- An: 净截面面积
- f: 钢材抗拉强度设计值

## 第11章 桁架

### 11.1 一般规定

11.1.1 桁架应进行杆件强度、稳定性和节点承载力计算。

11.1.2 桁架的高跨比应符合下列规定:

**平面桁架高跨比:**
- 屋架: 1/10 ~ 1/6
- 托架: 1/8 ~ 1/5
- 吊车梁桁架: 1/10 ~ 1/8

**空间桁架高跨比:**
- 网架: 1/15 ~ 1/20
- 网壳: 1/5 ~ 1/7

### 11.2 杆件设计

11.2.1 桁架杆件的长细比应符合下列规定:

**受压杆件:**
- 上弦杆: λ ≤ 150
- 腹杆: λ ≤ 200

**受拉杆件:**
- 上弦杆: λ ≤ 300
- 腹杆: λ ≤ 350

### 11.3 节点设计

11.3.1 桁架节点应满足下列要求:
- 节点板厚度不应小于6mm
- 焊缝应满足强度要求
- 螺栓连接应满足承载力要求

## 第12章 框架

### 12.1 一般规定

12.1.1 框架应进行整体稳定性和构件承载力计算。

12.1.2 框架柱的长细比应符合下列规定:
- 框架柱: λ ≤ 120
- 支撑框架柱: λ ≤ 150

### 12.2 梁柱连接

12.2.1 框架梁柱连接应满足下列要求:
- 刚性连接: 能传递弯矩和剪力
- 铰接连接: 仅传递剪力
- 半刚性连接: 部分传递弯矩

## 附录B 构造要求

### B.1 最小截面尺寸

B.1.1 钢构件的最小截面尺寸:
- 最小板厚: 4mm (室内), 6mm (室外)
- 最小角钢肢厚: 4mm
- 最小槽钢腹板厚: 5mm
- 最小工字钢腹板厚: 6mm

### B.2 焊缝构造

B.2.1 焊缝的最小尺寸:
- 角焊缝最小焊脚尺寸: 3mm (t≤4mm), 5mm (4<t≤6mm)
- 对接焊缝坡口角度: 60° (单面), 45° (双面)

### B.3 螺栓连接

B.3.1 螺栓连接的构造要求:
- 最小螺栓直径: M12
- 最小端距: 2d₀ (d₀为孔径)
- 最小中心距: 3d₀
- 最大中心距: 8t 或 150mm (取较小值, t为较薄板厚)
